## MSFCONSOLE

## Metasploit Architecture — Know This for CEH

## Exploit
The attack code that takes advantage of a vulnerability. Example: ms17_010_eternalblue exploits Windows SMB.

## Payload
Code that runs AFTER exploitation. Types: Singles (standalone), Stagers (small downloader), Stages (larger code pulled down by stager).

## Module
Reusable code components: Auxiliary (scanners), Exploits, Post (post-exploitation), Encoders, NOPs.

## Meterpreter
Advanced payload that runs in memory. Gives you a powerful shell with commands like sysinfo, hashdump, upload, screenshot.
